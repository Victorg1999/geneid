Promoter analysis
  • co-expressed genes clustering workflow: sequences in FASTA format input

    Protocol design:[short description]
    Input example: wf1-input.xml
    Outputs: outClustalwSlow, outClustalwFast, outTcoffee, outHmmbuild, outHmmsearch, outHmmpfam.

    Summary: Retrieves 10 AA sequences from UniProt and runs clustalwSlow; builds an HMM profile of
    the alignment (hmmbuild) and searches it against the previous sequences (hmmsearch).
    It also retrieves 3 DNA sequences from UniProt and runs clustalwFast and Tcoffee on them and runs
    hmmpfam on a protein sequence.
    Picture of this implemented workflow (using Taverna): PromoterAnalysisWorkflow_coexpressedGenesMode
    There is also a version available accepting a gene identifiers list, instead of a set of FASTA sequences, gene identifiers input workflow
