Analysis
  • GeneID v1.2:

    • runGeneIDGFF: Reads a DNA Sequence and returns GeneID predictions in GFF format.
      service specifications
      input xml, execute it using MOWServ.

    • runGeneID: Reads a DNA Sequence and returns GeneID predictions in HTML format.
      service specifications
      input xml, execute it using MOWServ.

    • translateGeneIDGFFPredictions: Reads a Nucleotide Sequence and Geneid predictions in GFF format
      and returns a collection of AminoAcid Sequences.

  • SGP2:

    • runSGP2GFF: being implemented.

  • MatScan:

    • runMatScanGFF: Reads a Nucleotide Sequence and returns a map of putative Transcription Factors binding sites in GFF format.
      You can select either Transfac (v6.4) or Jaspar motifs collections.
      service specifications
      input xml, execute it using MOWServ.

    • runMatScanGFFCollection: Reads a collection of Nucleotide Sequences and returns a collection of maps of putative binding sites in GFFF format.
      service specifications
      input xml, execute it using MOWServ.

  • Meta-alignment:

    • runMetaAlignment: Reads two binding sites maps in GFF format and returns the pairwise alignment of this maps in meta format.
      service specifications
      map1 xml and map2 xml, execute it using MOWServ.

    • runMetaAlignmentGFF: Reads two binding sites maps in GFF format and returns the pairwise alignment of this maps in GFF format.
      service specifications
      map1 xml and map2 xml, execute it using MOWServ.

    • runMultiMetaAlignment: Reads a collection of binding sites maps in GFF format and a collection of pairwise alignment.
      It generates the map pairs internally, and runs meta-alignment internally.

    • generateScoreMatrix: Reads a collection of pairwise meta-alignment output and returns a score matrix.

  • MEME v3.5:

    • runMemeText: reads a collection of generic sequences (it can be either AminoAcid sequences or Nucleotide sequences)
      and returns predicted MEME motifs in MEME Text output.
      service specifications
      input xml, execute it using MOWServ.

    • runMemeHTML: reads a collection of generic sequences (it can be either AminoAcid sequences or Nucleotide sequences)
      and returns predicted MEME motifs in MEME HTML output.
      service specifications
      input xml, execute it using MOWServ.

    • parseMotifMatricesfromMEME: read MEME Text output and returns the set of predicted MEME motifs as a text-formatted object.

Retrieval
  • Ensembl:

    • getUpstreamSeqfromEnsembl. Reads an object (an Ensembl gene identifier) and returns the upstream sequence of this given gene from Ensembl.

    • getUpstreamSeqfromEnsemblCollection. Reads a collection of objects (a set of Ensembl gene identifiers) and returns the collection of upstream sequences.

    • getOrthologousUpstreamSeqfromEnsembl. Reads an object (an Ensembl gene identifier) and returns the upstream sequence of all Ensembl predicted orthologues of this given gene.

Conversion
  • fromGenericSequencetoFASTA. Reads a GenericSequence object and converts it into FASTA format.

  • fromGenericSequenceCollectiontoFASTA. Reads a collection of GenericSequence objects and converts it into FASTA format.

  • fromFASTAtoDNASequenceCollection. Reads a set of DNA sequence in FASTA format and returns a collection of DNASequence objects.
    service specifications
    input xml, execute it using MOWServ.
