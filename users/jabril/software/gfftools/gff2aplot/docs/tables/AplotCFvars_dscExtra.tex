\idesc{\op{font} $\,=\,$ \pp{font-alias}  \pp{font-name} \hfill [ \bydef ]}
   { Record format for font special features is the following: font  font-alias  font-name where font-alias will serve to call this font in other customization variables section. As example, having font-alias set as palatino for a font-name defined as Palatino-Roman in the printer, you have to recall it as my\_palatino. }
%
\idesc{\op{color} $\,=\,$ \pp{color-alias}  \pp{cyan} \pp{magenta} \pp{yellow} \pp{black} \hfill [ \bydef ]}
   { Record format for color special features is the following: color  \pp{color-alias}  \pp{cyan} \pp{magenta} \pp{yellow} \pp{black} \pp{cyan}, \pp{magenta}, \pp{yellow} and \pp{black} correspond to each color amount that is required to produce the desired CMYK color. }
