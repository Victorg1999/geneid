Promoter analysis
  • co-expressed genes clustering workflow:
    input: the upstream sequences (FASTA format) of a set of co-expressed genes
    output: a gene clustering tree (picture in png format)
    description: cluster genes together based on upstream DNA sequence patterns in order to validate expression clusters
    Protocol design:[short description]

    A web page has been designed to execute it.
    The workflow has also been implemented using taverna software. You can download the workflow and load it in your local taverna installation. You must have the latest version of taverna (v1.3.1).
    Input document example: ENSG00000174697 upstream sequence and its orthologs
    Picture of this implemented workflow (using Taverna): PromoterAnalysisWorkflow_coexpressedGenesMode

    NB : There is also a version available accepting a gene identifiers list, instead of a set of FASTA sequences, gene identifiers input workflow
