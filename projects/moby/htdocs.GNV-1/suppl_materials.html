Supplementary material for manuscript,

"BioMoby services to support clustering of co-regulated genes based on similarity of promoter configurations".
 
 
  The co-expressed genes analysis pipeline was assessed on a set of genes that was extracted from Thompson et al., which reports the identification of a new module of co-expressed genes.

If you would like to repeat our analysis, you can use either our Web submission page or the Taverna software with the Workflow implemented with a list of gene identifiers input.
  • Input for the Data submission Web page:
    • The list of human genes
    • The 500 bp upstream sequences in FASTA format
       
  • Input for the Taverna Workflow:
    • The list of human genes
       
  • The parameter settings are the following:
    • MatScan: database=jaspar; threshold=0.86
    • TF-alignment: &alpha=0.5; &lambda=0.1; &mu=0.1
    • SOTA: distance=euclidean; resource_threshold=35
    • Multiple TF-alignment: &alpha=0.5; &lambda=0.1; &mu=0.1; Y=-10; c=100

     
  • Results
    • The results reporting page
    • A complete archive with the output of every steps.
    • The SOTA tree generated with GEPAS Tree visualization tool.
